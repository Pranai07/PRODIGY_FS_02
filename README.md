# 🔒 SecureAuth - Secure User Authentication System

## 📌 Overview

SecureAuth is a full-stack authentication system built using React, Node.js, Express.js, PostgreSQL, and Prisma ORM. It provides secure user registration, login, logout, JWT authentication, password hashing, and protected routes.

---

## 🚀 Features

- User Registration
- User Login
- Secure Password Hashing (bcrypt)
- JWT Authentication
- HTTP-only Cookies
- Protected Routes
- User Profile
- Logout Functionality
- Responsive UI
- Modern Dashboard
